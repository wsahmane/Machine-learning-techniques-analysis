# Applied Machine Learning Techniques

## 📌 Overview

This project explores core machine learning techniques applied to structured data to uncover patterns, relationships, and predictive insights.

It demonstrates the use of both supervised and unsupervised learning methods in a business-oriented analytical context.

---

## 🧠 Project Objectives

- Apply fundamental machine learning algorithms to real datasets  
- Identify patterns and relationships within structured data  
- Build predictive models and evaluate their performance  
- Compare different analytical techniques  

---

## 📂 Project Structure

The project is divided into multiple notebooks, each focusing on a specific method:

### 🔹 Clustering
- Grouping similar observations  
- Identifying hidden structures in data  
- Unsupervised learning approach  

### 🔹 Association Analysis
- Discovering relationships between variables  
- Market basket analysis techniques  
- Rule-based insights  

### 🔹 Regression
- Predicting numerical outcomes  
- Understanding relationships between variables  
- Model evaluation
